pwd: Nos muestra la ruta de carpetas en la que te encuentras ahora mismo.

mkdir: Nos permite crear carpetas (por ejemplo, mkdir Carpeta-Importante).

touch: Nos permite crear archivos (por ejemplo, touch archivo.txt).

git init = para iniciar el repositorio

git add + nombre del archiv = los pone listo para enviar con el commits

git rm --cached = borra los datos de la memoria ram 



git config = para configurar el git 

git config --global user.name + "nombre"

git config --global user.email + "correo del usuario"
